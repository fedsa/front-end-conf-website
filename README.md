# Front-end conference web site

## Local development

```
npm install
```

```
npx eleventy
```

Or build automatically when a template changes:
```
npx eleventy --serve --watch
```

Or in debug mode:
```
DEBUG=* npx eleventy
``

``gulp build & gulp watch`` 

``npx eleventy --serve --watch``
